# ImpactLedger Template

## Sobre o desafio
Template oficial do desafio ImpactLedger do Hackathon Web3 RESTIC 29.

## Objetivo
Construir uma solução baseada em blockchain capaz de registrar, validar e certificar ações de impacto social e ambiental.

## Exemplos de aplicação
- Certificação de voluntariado
- NFTs comunitários
- Registro de impacto ESG
- Dashboard auditável
- Plataforma de doações
- Sistema de reciclagem

## Tecnologias sugeridas
- Solidity
- Hardhat
- Sepolia
- Polygon Mumbai
- OpenZeppelin
- IPFS
- Ethers.js
- React

## Estrutura
/contracts
/frontend
/scripts
/test
/docs

## Como executar

### Instalar dependências
```bash
npm install
```

### Compilar contratos
```bash
npx hardhat compile
```

### Deploy
```bash
npx hardhat run scripts/deploy.js --network sepolia
```

## Requisitos mínimos
- Uso de blockchain
- Registro auditável
- Smart contract funcional
- Histórico verificável
- README funcional
- Vídeo-pitch
